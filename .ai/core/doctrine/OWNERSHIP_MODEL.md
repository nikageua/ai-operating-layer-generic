# Ownership Model

A shared operating layer needs explicit ownership for its durable truth.

## Roles

### Story owner
Responsible for local clarity inside one story.
May be a human or a main AI session.

### Task executor
Responsible for completing one scoped unit of work.
May be a human or sub-agent.
Does not own durable truth.

### Project curator
Responsible for deciding whether a lesson becomes project-shared truth.
This is a role, not necessarily one permanent person.

### Core curator
Responsible for deciding whether a lesson changes generic AIL doctrine.
This should be rarer and more conservative than project curation.

## Rule

Shared truth must always have an active curator role.
If nobody curates durable truth, the overlay becomes orphaned memory.
