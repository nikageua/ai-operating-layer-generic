# Stop and Ask

The agent should stop and ask for human input when continuing would create outsized risk.

## Triggers

1. About to change a contract consumed elsewhere.
2. About to introduce a new dependency or external integration.
3. About to make a one-way-door change.
4. Two consecutive reclassifications happened without convergence.
5. Scope has clearly grown beyond the original task or story.

## Rule
When one of these triggers fires, speed is no longer the priority. Human alignment is.
