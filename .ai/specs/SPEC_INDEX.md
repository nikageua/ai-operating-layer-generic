# Spec Index

Specs are the entry point for every feature. A feature must have an approved spec
before epics and tasks can be created from it.

## Draft
Features being actively written or refined with the AI agent.
- none

## Review
Specs complete enough for human review before architecture work begins.
- none

## Approved
Specs that are signed off. Epics and architecture decisions may now be created.
- none

## Implemented
Specs whose epics are all done.
- none
